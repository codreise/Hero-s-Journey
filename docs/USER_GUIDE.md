# Hero’s Journey: Короткий гайд для гравця

## Що це за гра

Hero’s Journey — це коротка хвильова RPG, де потрібно переживати хвилі ворогів, обирати нагороди, бити босів і поступово відкривати постійні покращення між забігами.

## Як запустити

1. Встановити залежності:

```powershell
npm install
```

2. Запустити гру:

```powershell
npm run dev
```

3. Відкрити в браузері:

```text
http://localhost:5173
```

## Базове керування

- WASD або стрілки — рух
- Пробіл або Z — удар
- P або Esc — пауза
- На телефоні — кнопки керування під ігровим полем

## Як проходить один забіг

1. Почати нову гру або продовжити збереження.
2. Очистити хвилю від ворогів.
3. Обрати одну нагороду після хвилі.
4. Кожна п’ята хвиля — бій з босом.
5. Після завершення забігу отримати есенцію для постійних покращень.

## На що звертати увагу

- Не тримай зілля до останнього моменту.
- Якщо бос стріляє, не стій на місці.
- Після нагороди краще підсилювати те, чого зараз не вистачає: HP, атаку або виживання.

## Де зберігається прогрес

- У браузері поза Telegram — в localStorage.
- У Telegram — через локальний save-сервер, якщо він запущений.

## Що зараз є в грі

- хвилі звичайних ворогів
- боси
- нагороди між хвилями
- пауза і продовження
- мета-прогрес і покращення табору
- мобільне керування

## Якщо щось не працює

- Перезапусти фронтенд через `npm run dev`.
- Для Telegram-збережень окремо запусти `npm run server`.
- Якщо сторінка не оновилась, перезавантаж вкладку.
